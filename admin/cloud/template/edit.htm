根目录:
  • 设置根目录，对于某些网盘需要设置根目录。注意：目录名要区分大小写
  • 比如百度网盘只能访问apps/合作网站名称/下的文件
  • 比如：/apps/dzzoffice,结尾不加/
  • 详细的设置和说明可以参照相关的api开发文档
